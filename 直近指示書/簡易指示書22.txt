マウスの位置からy座標下にラケットがあるが、それだと画面上で見たときにマウスとラケットの位置が重ならず、可動域が極端に狭くなってしまっている。
マウスとラケットの位置が視点上で重なるように、そして、ラケットのy座標は固定し、左クリックしてない間はラケットが動かないようにして。
あと、右クリック長押しでラケットを手前に引くと、
data modify entity @e[tag=rake] rotation set value {angle:1.570,axis:[0,0,-1]}
のangleが少し下がり、下回転打球となる。また、
右クリック長押しでラケットを奥に動かすと
data modify entity @e[tag=rake] rotation set value {angle:1.570,axis:[0,0,-1]}
のangleが少し上がり、上回転打球となる。
